# Divan Adabı

Divan'da söz herkesin, saygı esastır.

- Fikre itiraz et, kişiye değil. Gerekçesiz ret yok, kanıtlı eleştiri makbul.
- Yeni gelen misafirdir: ilk PR'a sabırla, yol göstererek yaklaş.
- Taciz, hakaret, ayrımcılık ve kişisel veri ifşası Divan'dan ihraç sebebidir.
- Anlaşmazlıkta hakem repo sorumlusudur; kararlar issue üzerinde şeffaf verilir.

İhlal bildirimi: repo sorumlusuna GitHub üzerinden özelden ulaşın veya
"conduct" etiketli issue açın.
