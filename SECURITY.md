# Güvenlik Politikası

Skill'ler ajan davranışını yönlendirir; kötü niyetli bir skill tehlikelidir.

- Güvenlik açığını **herkese açık issue ile BİLDİRME**.
- GitHub "Security Advisories" (Report a vulnerability) üzerinden özel bildir.
- Kapsam: prompt injection kalıpları, zararlı komut önerileri, lisans ihlali.
- Hedef yanıt süresi: 72 saat.
